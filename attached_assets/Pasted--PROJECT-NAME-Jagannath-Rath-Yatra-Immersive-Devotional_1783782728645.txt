# PROJECT NAME
Jagannath Rath Yatra – Immersive Devotional Experience

# OBJECTIVE

Create an ultra-premium devotional website centered around the Jagannath Rath Yatra festival. The experience should feel sacred, elegant, cinematic, and respectful—not game-like or cartoonish.

The homepage combines:

1. A fully interactive 3D Jagannath Rath (left side)
2. Live YouTube Darshan (right side)
3. Premium sponsor placements
4. Festival ambience
5. Mobile-first responsive layout

The emotional goal should be:

"Wow...this feels like participating in Rath Yatra."

Use modern premium web design similar to Apple + luxury museum websites, but infused with Indian devotional aesthetics.
